HTML Cheatsheet

HTML Elements

An HTML element is a piece of content in an HTML document and uses the following syntax: opening tag + content + closing tag.

<h1>Hello World</h1>

Tag Name Description
<body> Body Element Represents the content of an HTML document
<div> Div Element Used as a container that divides an HTML document into sections and is short for “division”
<span> Span Element An inline container for text and can be used to group text for styling purposes
<h1> - <h6> Heading Elements Ordered from the highest level <h1> to the lowest level <h6>
<p> Paragraph Element Contains and displays a block of text
<br> Line Break Element Creates a line break in text and is especially useful where a division of text is required. This is a self-closing tag
<em> Emphasis Element Emphasizes text and browsers will usually italicize the emphasized text by default
<strong> Strong Element Highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default
<li>, <ol>, <ul> List Item Element Creates list items inside. Can be ordered <ol> or unordered <ul>
<ol> Ordered List Element Creates a list of items in sequential order. Each list item appears numbered by default
<ul> Unordered List Element Used to create a list of items in no particular order. Each individual list item will have a bullet point by default
<img> Image Element Embeds images in documents and is a self-closing tag. The src attribute contains the image URL and is mandatory
<video> Video Element Embeds a media player for video playback. The src attribute will contain the URL to the video
