HTML Elements

What is an HTML element?

An HTML element is a piece of content in an HTML document and uses the following syntax: opening tag + content + closing tag.

<h1>Hello World</h1>

Tag Name Description
<h1> - <h6> Heading Elements Ordered from the highest level <h1> to the lowest level <h6>
<li>, <ol>, <ul> List Item Element Creates list items inside. Can be ordered <ol> or unordered <ul>
